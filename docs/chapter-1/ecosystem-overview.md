## Ecosystem Overview

### Blockchain Technology and Consensus

Consensus mechanisms are how blockchains agree on the state of the network. Common types include:

- **Proof of Work (PoW)**: Miners solve puzzles to validate transactions (e.g., Bitcoin).
- **Proof of Stake (PoS)**: Validators stake tokens to secure the network (e.g., Ethereum 2.0).
- **Proof of Transfer (PoX)**: Unique to Stacks, PoX anchors Stacks blocks to Bitcoin blocks, leveraging Bitcoin’s security.

### What is Stacks?

Stacks is a layer-1 blockchain that brings smart contracts and decentralized apps to Bitcoin. It does this without modifying Bitcoin itself. Instead, it uses PoX to settle transactions on Bitcoin while enabling expressive smart contracts written in Clarity — a secure, predictable language.

Stacks enables:

- Smart contracts secured by Bitcoin
- DeFi and NFTs on Bitcoin
- On-chain identity and reputation systems
- Bitcoin-native DAOs and apps

### Stacks Architecture and Security

Stacks operates independently but is anchored to Bitcoin. Here’s how it works:

- **Clarity smart contracts**: A decidable language that avoids runtime surprises. You can know exactly what a contract will do before running it.
- **Stacks blockchain**: Handles smart contract execution and state changes.
- **PoX consensus**: Connects Stacks blocks to Bitcoin blocks, giving Stacks apps Bitcoin-level finality.

This architecture means that Stacks apps inherit Bitcoin’s security while enabling features Bitcoin alone cannot support.
